# blcaptain-ppt-skill — Product

> 一句话：把一个目标 / 文章 / 数据，变成有判断、有节奏、第一眼专业的**单文件 HTML 演示**——7 套体系锚定人格，**好看与诚实都机器强制**。
> 能力总览见 [README](README.md)，Agent 工作流见 [SKILL](SKILL.md)。

## 是什么 / 给谁

一个面向演示创作者的 deck 生成 Skill（Codex / Claude Code / Cursor / Gemini CLI…）。给：做技术战报、产品拆解、行业观察、路演 / 发布会、数据复盘、思想 / 品牌 keynote 的人——任何需要「内容 → 专业演示」的人。

工作形态是**智能驱动**：Agent 读懂内容、判一套人格、做导演判断，然后在被验证过的版式里填内容——**不让 AI 自由画版面**（AI 在自由版面上平庸，给它被验证过的骨架，任务从「设计」降级成「填充」，成品稳定性立刻上来）。理解归 Agent，确定性渲染 + 质量校验归代码。

## 解决什么问题

市面 AI PPT 一眼假——大色块 + emoji + 紫蓝橙粉渐变 + 居中堆叠，**「做的是网页，不是演示」**。还有：数据随手编、配图一眼 AI 插画、暗底投影看不清、开发 / demo 信息漏进成品。

我们把这些变成**可执行的约束**——字号阶、留白比例、对比度阈值、断行规则、配色角色、暗底投影地板、反伪造纪律，**落到代码里全是常量 + 机检门**。这些常量与门才是真正的护城河。

## 7 套视觉人格（护城河 = 深度而非数量）

每套锚定一个**公认设计体系**，有一句话灵魂，不是换色皮肤：

- **技术崇高 Technical Sublime** — Vercel / Linear / Stripe 工程冷光 + iso-compute 算力等值场。技术战报 / 算力 / AI。
- **构成主义 Constructivism** — Rodchenko / El Lissitzky 红场分幕 + 非对称张力。宣言 / 号召。
- **信息·数据 Information Design** — Tufte / FT / Pudding 图表语汇，数据即论据。数据 / 报告。
- **判断震慑 Luxury Minimalism** — Didone 裸色 + 留白即定价。高端品牌 / 判断。
- **新造型 De Stijl** — Mondrian / Vignelli 几何无衬线 + 浊三原色。标准 / 秩序。
- **编辑主义 Editorial** — 杂志编辑传统 + 唯一衬线正文。深度观察 / 长文。
- **演示禅 Zen** — 阴翳 × MUJI 空 + 暗墨「第二种黑」。思想 / 品牌 keynote。

## 机器强制品味 OS（独家能力 = 结构不可复制）

token + WCAG 对比度门 + 间距刻度门 + P0–P2 校验 + 32 维审计 = **codified taste-OS**。不是「建议」，是**过不了不交付**。配合**反伪造机检**（数据 / logo / 截图 / 来源）与**本地字体覆盖机检**（断网不崩），构成竞品普遍没有的「机器强制下限」。

## 能力边界（不接什么 = 产品定义）

明确不接：大表格 / 培训教材 · 法律 / 医疗 / 金融合规文件 · 多人协作编辑的传统 PPT · 纯文本摘要 · 复刻某品牌官方发布会视觉 · 任何编造真实数据 / logo / 截图。碰到这些，开头就告诉用户「你可能想用别的工具」——**一个什么都能做的 Skill 最后通常什么都做不好。**

## vs 上游优秀项目（领先在哪，不复制）

研究借鉴业界优秀项目的产品化流程 / 固定版式思路 / 渲染链路 / 验证意识，但**不复制其设计、代码、模板、CSS、资产、配色、布局 ID 或视觉身份**（clean-room 独立实现）。领先点：

① **机器强制一致**（WCAG / 间距 / 32 维审计——别家靠运气）
② **7 套体系锚定深度**（别家靠模板数量堆广度）
③ **反伪造诚信**（成文 + 机检）
④ **本地字体不崩**（别家 CDN 断网 / 墙内崩）
⑤ **暗底投影可读**（AAA 地板·公开演讲真看得清）
⑥ **零依赖**（只用 Node 内置·git clone 即用）

对你的 Agent 说：「帮我安装这个 Skill：`github.com/dososo/blcaptain-ppt-skill`」，然后「用 blcaptain 把这段内容做成演示」。详见 [README](README.md) 与 [SKILL](SKILL.md)。
